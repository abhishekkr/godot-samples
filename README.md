## godot-samples

> none of the tutorials are followed to the Tee; wherever my overall experience from usage and other tutorials differ.. the implementation gets few tweaks

* [Creeper Dodger](https://github.com/abhishekkr/godot-creeper-dodger): following 2d game from official doc
* [Creeper Dodger 3D](creeper-doder-3d/): following 3d game from [official doc](https://docs.godotengine.org/en/stable/getting_started/first_3d_game/index.html)

* [Brackeys' 2D](brackeys-2d/): following 2D game tutorial from [YT](https://www.youtube.com/watch?v=LOhfqjmasi0); used Hurtboxes, Globals and Control

* [Road Hoppers](./road-hopper): 2D game on `crossing the road` from [ClearCode's YT tutorial](https://www.youtube.com/watch?v=TLG2yVpLDT8)
* [Like a Metroid](./like-a-metroid): 2D metroid like game from [ClearCode's YT tutorial](https://www.youtube.com/watch?v=TLG2yVpLDT8)
* [Space Shooter 3D](./space-shooter-3d): 3D Space Shooter game from [ClearCode's YT tutorial](https://www.youtube.com/watch?v=TLG2yVpLDT8)

* [GDQuest's Vampire Survior Style](./gdquest-vampire2d): 2D game for quick shooter style from [GDQuest's Tutorial](https://www.youtube.com/watch?v=GwCiGixlqiU)
